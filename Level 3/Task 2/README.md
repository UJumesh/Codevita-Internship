# Level 3 Task 2 - Real Time Skill Notification System

## Overview
A real-time skill management notification system built using WebSocket technology.

## Tech Stack

Frontend:
- React.js
- Socket.io Client
- CSS

Backend:
- Node.js
- Express.js
- Socket.io

## Features

✅ Real-time notifications  
✅ Multiple users connection tracking  
✅ Online users count  
✅ Skill broadcast system  
✅ Live updates without refresh  

## How to Run

Backend:

npm install
npm run dev


Frontend:

npm install
npm start
